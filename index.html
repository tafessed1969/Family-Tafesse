The Web

The Web (or World Wide Web) is a collection of documents and other files. Most are made using HTML.

What is HTML?

HTML stands for Hypertext Markup Language. It defines the structure or layout of a web page.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>
